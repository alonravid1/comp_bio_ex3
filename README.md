    <h1>Neural Network with Genetic Algorithm</h1>

    <h2>Programs</h2>

    <p>There are two separate problems addressed in this project, each requiring a different program.</p>

    <h3>Buildnet0</h3>

    <p>Buildnet0 program is designed to build a neural network for the first legality problem. It takes a learning file and a test file as input and outputs the structure definitions of the network and the weights of the network.</p>

    <p>To run buildnet0 program:</p>

    <ol>
        <li>Clone the repository: <code>git clone https://github.com/alonravid1/comp_bio_ex3.git</code></li>
        <li>Place the training and test data files in the same folder as the buildnet0 program. Name the files <code>training_set0.txt</code> and <code>test_set0.txt</code>.</li>
        <li>Double-click on the <code>buildnet0.exe</code> file.</li>
        <li>When prompted, choose whether to see information about the program (y/n).</li>
        <li>Enter the path to the training data file or press Enter to use the default file name.</li>
        <li>Enter the path to the test data file or press Enter to use the default file name.</li>
        <li>Wait for the program to run. The running time will be displayed.</li>
        <li>Press Enter to exit the program.</li>
    </ol>

    <p>The output file <code>wnet0.txt</code> will be generated in the same folder as the buildnet0 program.</p>

    <h3>Runnet0</h3>

    <p>Runnet0 program takes the output file <code>wnet0.txt</code> generated by buildnet0 and a data file of 20,000 binary strings as input. It predicts the legality of each string based on the learned network and outputs the predictions to a text file.</p>

    <p>To run runnet0 program:</p>

    <ol>
        <li>Clone the repository: <code>git clone https://github.com/alonravid1/comp_bio_ex3.git</code></li>
        <li>Place the <code>wnet0.txt</code> file generated by buildnet0 and the data file in the same folder as the runnet0 program. Name the data file as desired.</li>
        <li>Double-click on the <code>runnet0.exe</code> file.</li>
        <li>Press Enter to start the program.</li>
        <li>Wait for the program to complete.</li>
        <li>The predictions will be saved in the file <code>predictions0.txt</code> in the same folder as the runnet0 program.</li>
    </ol>
